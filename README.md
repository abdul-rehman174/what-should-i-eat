# What Should I Eat?

A fun little spinner for when you and the hostel bruhs **can't decide what to eat**. 🎰

Built with plain HTML/CSS/JS — no build step, no dependencies. Just open `index.html`.

## ✨ Features
- 🎲 Slot-machine spin animation that slows down for suspense
- 🍛 Real **Pakistani hostel menu** — daal chawal, mess biryani, halwa puri, and the legendary **"Broke 💸"** mode
- 🚫 Never suggests the same thing twice in a row
- ⌨️ Press `Space` or `Enter` to spin again
- 📱 Works on phone and desktop

## 🚀 Run it locally
Just double-click `index.html`, or:
```bash
open index.html
```

## 🌐 Deploy
This is a static site — deploys to Vercel, Netlify, or GitHub Pages instantly.
